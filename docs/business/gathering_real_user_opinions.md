# Gathering real user opinions

## [Perplexity AI](https://www.perplexity.ai/)

Since it summarizes information based on online articles, it's a
recommended method for getting a rough analysis or a starting point for
deeper investigation. You can simply enter a query like, "Summarize user
complaints about XXXX."

## Online Message Boards

- [Reddit - The heart of the internet](https://www.reddit.com/)
- [ビュアデモ検索](https://www.kyodemo.net/sdemo/s)
- [5ちゃんねるスレタイ検索](https://ff5ch.syoboi.jp/)

An approach to searching message boards. Traditional, yet potentially
useful for gathering real user opinions.
