# Key Takeaways

## インサイト中心の成長戦略

- インサイトは日常の実務から
    - 通常業務の中で発見する: 特殊な調査ではなく、日々の仕事で得る「違和感や解釈」が戦略の起点になる
    - 画期的なイノベーションは不要: 「誰も見たことがないもの」を目指さず、儲かっている先行者の欠点を突けば十分勝てる
- **「考えて作って売る」下積みの徹底**
    - 能力獲得には実務経験が必要: 座学ではなく、自分を現場に置き、事業を動かすことでしか能力は身につかない
    - とにかく販売を経験する: 下積み期間は収益より経験を重視し、拙い商品でも早く「作って売る」ことで、顧客ニーズの解像度を爆速で高める

## おもろい話し方 芸人だけが知っているウケる会話の法則

- **大前提: すべてを取り入れる必要はなく、自分に馴染むものだけを採用する**
    - しっくりきたポイント: **「フリオチ」「たとえ」**
- お笑いタイプが同卓の人と被ってうまく話せなくなることはよくあるため、**「MCツッコミ」と「むじゃきボケ」を使い分ける**のが大切
- 「話しやすさ」の極意
    - **中見ゼロの発言を増やす**: 「電波悪い」が会話のハードルを下げる
    - **聞く時はキーワード・テーマ探し**: リアクションをしっかり取った上で、誰かの話のつながりで話す
- 「話し方」の黄金律
    - **フリオチ**: 「オチ=言いたいこと」からの逆算でフリを考える
    - **3つのスパイス**: 「擬音で臨場感」「普通との比較」「自分の感情」
    - **スロースタート**: 出だしは「低いトーンでゆっくり」
- 「返し」の技術
    - **たとえ**: 「○○モンスターやんけ」といった造語もあり
    - **ツッコミは短く**: 「いや、」から入り、誰もが最初に抱く違和感を一言で突く
    - **逆マウンティング**: イジリをより強く肯定する

## すぐやる脳

- やる気がなくてもやり始めればやる気が出る(作業興奮)
- 完璧主義ではなく、見切り発車する
- 目標を小さく分割して成功体験を積み上げる
    - 例: デイリー、ウィークリー、マンスリーで分ける
- 内発的モチベーションが大切
    - **ワクワクしながら楽しむことは努力に勝る**
- 部屋を掃除・整理整頓することがやる気につながる

## 嫌われる勇気

- 傷つかないために、人はできない理由を作り出す
- 禁止エリアへ一歩踏み出すことが大切
    - 失敗すればいい
- 他人はお前のことなんか興味ない
- 他人の期待を満たすために生きてはいけない
    - 自分の課題に集中する(課題の分離)
- 人生は100%自分で決めて、自分で創っていくもの

## 神モチベーション 「やる気」しだいで人生は思い通り

- ハイモチベーションに頼ってはいけない
- ギャップモチベーションを使いこなすことが鍵
    - なりたい理想(目標)を強くイメージする

## 解像度を上げる

- 大前提、解像度を上げることに時間をかけすぎないこと
- 広くて深いロジックツリーが作れる状態になる
    - 一番大事なセンターピンを特定して抑える
        - 時代やトレンドを考慮して特定する
        - 歴史や過去の失敗を確認する
- 「深さ」を上げるには、とにかく行動してから考えることが大事
- 図に起こすことで頭の中の「構造」が整理される
- 解像度が高くなると、以下のテンプレートが簡潔に埋められるはず
    - 顧客の課題と競合との差を意識すること

```
<状況>という状況で
<課題>という課題を持つ
<対象顧客>向けの
<製品/サービス名>という
<製品/サービスジャンル>です。
これには<利点>という利点があります。
<既存の代替え品・競合>とは違い
<差別化要素>が備わっています。
```
