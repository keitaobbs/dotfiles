# Key Takeaways

## 嫌われる勇気

- 傷つかないために、人はできない理由を作り出す
- 禁止エリアへ一歩踏み出すことが大切
    - 失敗すればいい
- 他人はお前のことなんか興味ない
- 他人の期待を満たすために生きてはいけない
    - 自分の課題に集中する(課題の分離)
- 人生は100%自分で決めて、自分で創っていくもの

---

## すぐやる脳

- やる気がなくてもやり始めればやる気が出る(作業興奮)
- 完璧主義ではなく、見切り発車する
- 目標を小さく分割して成功体験を積み上げる
    - 例: デイリー、ウィークリー、マンスリーで分ける
- 内発的モチベーションが大切
    - 楽しい・面白いとワクワクしながら取り組めているか？
- 部屋を掃除・整理整頓することがやる気につながる

---

## 神モチベーション 「やる気」しだいで人生は思い通り

- ハイモチベーションに頼ってはいけない
- ギャップモチベーションを使いこなすことが鍵
    - なりたい理想(目標)を強くイメージする

---

## 解像度を上げる

- 大前提、解像度を上げることに時間をかけすぎないこと
- 広くて深いロジックツリーが作れる状態になる
    - 一番大事なセンターピンを特定して抑える
        - 時代やトレンドを考慮して特定する
        - 歴史や過去の失敗を確認する
- 「深さ」を上げるには、とにかく行動してから考えることが大事
- 図に起こすことで頭の中の「構造」が整理される
- 解像度が高くなると、以下のテンプレートが簡潔に埋められるはず
    - 顧客の課題と競合との差を意識すること

```
<状況>という状況で
<課題>という課題を持つ
<対象顧客>向けの
<製品/サービス名>という
<製品/サービスジャンル>です。
これには<利点>という利点があります。
<既存の代替え品・競合>とは違い
<差別化要素>が備わっています。
```

## 誰でもできるのに9割の人が気づいていない、お金の生み出し方

- 人は欲求にお金を払う
- 多くの人に買ってもらわなくてもいい
    - 1%にしかニーズがなくても日本で100万人にニーズがある
- すべての人に認められなくていい
    - 全ての意見を取り入れるのはダメ
- 100%の結果を保証しなくてよい
- 差別化しなくていい
    - 顧客のためではなく差別化が目的になってしまう
